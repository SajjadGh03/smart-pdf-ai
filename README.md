# Smart PDF AI

A simple AI-powered PDF question answering system using RAG (Retrieval-Augmented Generation).

## Features
- Ask questions from PDF documents
- AI-generated answers using LLM
- Insight Mode:
  - Summary
  - Key points
  - Suggested questions
  - Ambiguity detection

## Tech Stack
- Python
- Streamlit
- FAISS
- Sentence Transformers
- OpenAI API

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
